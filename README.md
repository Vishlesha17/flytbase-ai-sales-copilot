# FlytBase AI Sales Copilot

## Overview
This project is an AI-powered Inbound BDR Sales Copilot built for the FlytBase assignment using n8n and Google Gemini.

## Features

- Lead Qualification (BANT)
- Company Research
- Industry Analysis
- FlytBase Use Case Mapping
- Discovery Questions
- Risk Analysis
- Opportunity Scoring
- Next Best Action
- Personalized Follow-up Recommendations

## Tech Stack

- n8n
- Google Gemini API
- AI Agent
- JSON Workflow

## Workflow

Trigger
↓
Lead Input
↓
AI Agent (Gemini)
↓
Lead Qualification
↓
Research
↓
Use Cases
↓
Discovery Questions
↓
Recommendations
